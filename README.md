
# MATLAB Learning

This repository documents my journey of learning MATLAB through progressively challenging engineering problems rather than isolated programming exercises.

## Objectives

- Develop strong MATLAB programming skills
- Apply MATLAB to energy systems and battery energy storage systems (BESS)
- Learn engineering modeling, simulation, data analysis, and control
- Build projects that reflect real engineering workflows

## Learning Methodology

Each project follows the same engineering process:

1. Engineering Problem
2. Reading Sprint
3. System Understanding
4. Design Review
5. MATLAB Implementation
6. Testing & Validation
7. Improvements
8. Reflection
